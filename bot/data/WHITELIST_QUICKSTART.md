# 🚀 Быстрый старт: `/add_server_for_whitelist`

## 📋 Что это?

Команда для автоматической настройки **BOT через whitelist-туннель** для обхода блокировок.

---

## ⚡ Использование

### 1. Подготовьте 2 сервера

**Местный (Россия):**
- ОС: Debian 11
- RAM: 2GB
- IP: например, `185.200.100.1`
- Пароль: например, `MyPass123`

**Зарубежный (Нидерланды):**
- ОС: Debian 11
- RAM: 2GB
- IP: например, `45.100.200.2`
- Пароль: например, `EuPass456`

### 2. Отправьте команду в бота

```
/add_server_for_whitelist 185.200.100.1 MyPass123 Москва 45.100.200.2 EuPass456 Amsterdam
```

### 3. Дождитесь настройки (6-12 минут)

Бот выполнит 5 шагов:
1. 🔍 Сканирование подсети для SNI домена
2. 🔄 Установка 3X-UI на зарубежный сервер
3. 🔄 Установка 3X-UI на локальный сервер
4. 🔒 Настройка VLESS туннеля
5. 💾 Добавление серверов в базу

### 4. Проверьте результат

```
✅ Whitelist-туннель успешно настроен!

📍 Местный сервер: 185.200.100.1 (Москва)
🌍 Зарубежный сервер: 45.100.200.2 (Amsterdam)
🎭 SNI маскировка: www.microsoft.com
📊 Панель: http://185.200.100.1:28308
```

```bash

/analytics
```

---

## ⚠️ Важно

- Пароли должны содержать **только буквы и цифры** (a-zA-Z0-9)
- Серверы должны быть **чистыми** (без установленных панелей)
- Зарубежный сервер должен иметь **чистый IP** (не в блокировках)
